# TMA

Welcome! Here you can find the relevant resources for contributing to the TMA page. The TMA page is constantly being developed. 

The TMA page is where we store all the info pertaining to operations within an Aerodrome's Terminal Manouvering Area.

You can find the relevant formats in the "TMA Format" section. It is important that you follow these formats. The reason for this is to make sure that each page has an equal amount of info.

Thank you!!
