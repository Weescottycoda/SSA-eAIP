# Aerodromes

Welcome! Here you can find the relevant resources for contributing to the Aerodromes page. The Aerodromes page is constantly being developed. 

The Aerodromes page is where we store all the info pertaining to operations within an Aerodrome's CTR.

You can find the relevant formats in the "Aerodrome Formats" section. It is important that you follow these formats. The reason for this is to make sure that each page has an equal amount of info.
