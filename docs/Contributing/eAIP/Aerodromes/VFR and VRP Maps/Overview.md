# Overview

Congratulations! By now you have very likely begun coding your first aerodrome page. Whether it be Togo or FASA, you may need to create 2 types of maps: visual flight rules (VFR) and visual reference points (VRP).

Furthermore, should you have any remaining questions, we strongly encourage you to ask in the VATSSA Community Discord server; a link can be found [here](https://community.vatsim.net/).
