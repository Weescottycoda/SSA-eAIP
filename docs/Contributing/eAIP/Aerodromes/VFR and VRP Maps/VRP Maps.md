# VRP Maps

## What is a VRP Map?

Certain aerdromes have Visual Reference Points within their CTRs. VRPs can be difficult to visualise for both the pilot and the Controller. To combat this we create VRP Maps.

VRP Maps are used for VFR Maps; therefore, it is really important that all the VRPs are displayed. A controller may require an aircraft to route to or report a certain VRP within their CTR.

## Creating VRP Maps

### Guidlines

- VRP Maps must be based on official sources. 

- The only copyright-free sources are from the real-world AIP, Government Sources or your own designed maps.

- VRP Maps must include all VRPs within the CTR (Clearly labeled).

### Personal VRP Maps

- The best way to create your own maps is through Google Earth.

- You can find the link to make custom Google Earth maps [here](https://earth.google.com/web/@0,0,0a,22251752.77375655d,35y,0h,0t,0r/data=CgRCAggBMikKJwolCiExaGg5VUlYdkk3T1ZLLUFVcWM0Ny1TUWFPTnJVdHFSOFogAToDCgEwQgIIAEoICMuy99gDEAE)

- Add a pin on the runway. Use a red pin with a plane icon. Use medium size.

- Add a pin for each VRP. Small size, Yellow and no icon.

- Now screenshot and use Canva to add labels. 

- Now use an image enhancer. You can now add it to your eAIP page and use it for STD VFR routings.
