# Pilot Briefings

Welcome! Here you can find the relevant resources for contributing to the Pilot Briefings page. We are always looking to create new pilot briefs.

The Pilot Briefs page is what we provide to pilots looking to fly within VATSSA. Pilot briefs contain the most relevant info. 

We also use the briefs for large-scale events such as Cross Africa (CAF) and Cross the Pond (CTP).

You can find the relevant formats in the "Pilot Brief Format" section. It is important that you follow these guides. The reason for this is to make sure that each brief has an equal and relevant amount of info.

Thank you!!
